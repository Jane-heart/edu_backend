# 在线教育平台后端开发指南

## 项目介绍

本项目是一个在线教育平台的后端开发，提供了课程管理、历史记录管理、推荐管理、建议管理、证书管理、小组管理、评论管理等功能。

## 快速启动

1. 克隆本项目到本地
2. 使用IDE打开项目
3. 配置数据库连接信息
4. 运行项目
